# Demo Questions

Use these questions to test the route, retrieval, and constrained answer display.

## Paper Questions

1. What method does the paper propose?
2. What experiments are reported in the paper?
3. What datasets or benchmarks are mentioned in the paper?
4. What limitations does the paper discuss?
5. 论文的主要结论是什么？

## Repository Questions

1. Which dependencies are listed in requirements.txt?
2. Where is the main implementation code in the repository?
3. What does the README say about running the project?
4. Which configuration files are present in the repository?
5. 仓库 README 说明了什么？

## Cross-Source Checks

1. Does the repository implement the method described in the paper?
2. Compare the paper method with the GitHub code implementation.
3. Can the repository reproduce the paper experiments?
4. Are the dependencies consistent with the paper's described framework?
5. 论文方法和仓库实现是否一致？
